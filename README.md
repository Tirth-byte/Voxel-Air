# Voxel Air

**"Build in 3D with your bare hands—right in the browser."**

[**🚀 Live Demo**](https://voxelair.netlify.app)

Voxel Air is a futuristic, gesture-controlled 3D modeling environment powered by AI. Using advanced hand tracking (MediaPipe) and WebGL (Three.js), it allows users to pinch, grab, and manipulate digital voxels in mid-air through a premium "Spatial Computing" interface.

Designed to be 100% static and deployment-ready, it runs flawlessly on Netlify with no backend dependencies.

---

## ✨ Features

- 🖐️ **AI Hand Tracking**: Real-time skeletal tracking using MediaPipe.
- 🧊 **Spatial Editing**: Pinch to draw, fist to grab, palm to rotate.
- 🎨 **Premium UI**: MacOS-style glassmorphism with neon "Iron Man" aesthetics.
- 📱 **Phone Mode**: Optimized FOV and interaction for mobile listeners.
- ⚡ **Performance**: 60fps optimized loop, ensuring smooth tracking on low-end devices.

---

## 🛠 Tech Stack

- **Three.js** – High-performance 3D rendering.
- **MediaPipe Hands** – Machine Learning hand tracking in-browser.
- **Vanilla JS (ES6+)** – Clean, modular, framework-free logic.
- **HTML/CSS** – Semantic structure with modern CSS animations.

---

## 🚀 Deployment

This project is configured for **Netlify**.
- `netlify.toml` handles SPA routing and redirects.
- `index.html` is the single entry point.
- Zero server-side code required.
